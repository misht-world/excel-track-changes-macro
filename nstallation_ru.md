# Подробное руководство по установке

## Быстрая установка

1. **Скачайте макрос** из этого репозитория
2. **Откройте ваш Excel файл** где нужно отслеживать изменения
3. **Включите вкладку Разработчик** (если еще не включена):
   - Файл → Параметры → Настройка ленты
   - Отметьте "Разработчик" в правой панели

## Пошаговая установка

### Способ 1: Прямое копирование
1. Нажмите `Alt + F11` для открытия редактора VBA
2. В обозревателе проектов дважды щелкните `ThisWorkbook`
3. Удалите существующий код
4. Скопируйте содержимое из `TrackChangesMacro.vba`
5. Вставьте в окно кода
6. Закройте редактор VBA (`Alt + Q`)
7. Сохраните книгу в формате `.xlsm`

### Способ 2: Импорт модуля
1. Откройте редактор VBA (`Alt + F11`)
2. Щелкните правой кнопкой `ThisWorkbook` → Вставить → Модуль
3. Скопируйте и вставьте код
4. Сохраните и закройте

## Тестирование макроса

1. Откройте сохраненный `.xlsm` файл
2. Измените любое значение ячейки - оно должно стать синим
3. Верните обратно к исходному - должно стать черным
4. Протестируйте с формулами и на разных листах

## Решение проблем

**Макрос не работает?**
- Убедитесь что файл сохранен как `.xlsm`
- Включите макросы при открытии файла
- Проверьте настройки центра управления безопасностью

**Проблемы с производительностью?**
- Макрос отслеживает все используемые ячейки
- Для очень больших книг рассмотрите ограничение диапазона

**Цвета не обновляются?**
- Ручное форматирование переопределяет цвета макроса
- При необходимости очистите ручное форматирование
